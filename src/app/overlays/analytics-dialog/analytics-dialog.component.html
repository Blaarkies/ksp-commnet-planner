Anonymous Usage Tracking
This application tracks anonymous usage data. Simple actions are be recorded with anonymized data, before being sent to the server. Typical actions include:
  • Dragging a planet
  • Adding checkpoints in the Delta-V calculator
  • Adjusting a craft's antennae type or count
  • Using keyboard shortcuts such as Tab to focus planets
  • etc.

This data provides me with a heat-map indicating which types of features should be prioritized.

No user data will ever be collected for this.
