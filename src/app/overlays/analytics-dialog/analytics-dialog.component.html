Anonymous Usage Tracking
This application tracks anonymous usage data. Actions such as the following will be recorded, have user data anonymized, and then sent to the server.
  • Dragging a planet
  • Using the Delta-V calculator
  • Adjusting a craft's antennae count
  • etc.

This data provides me with a heat-map indicating which features are most often used, and this optimizes where effort is prioritized.

No user names, craft names, planet names, or browser information will ever be collected for this.
