Anonymous Usage Tracking
This application tracks anonymous usage data. Simple actions are recorded as anonymized data before it is sent to the server. Typical actions include:
  • Moving a planet
  • Adding checkpoints in the Delta-V calculator
  • Adjusting a craft's antennae type or amount
  • Using keyboard shortcuts such as Tab to focus planets
  • etc.

This data helps me determine which features should be prioritized.

No user data will ever be collected in anonymized data.
